# Progress Bar

## What this spell does
Displays a horizontal progress bar whose width changes based on a value.

## When to use
- Loading states
- Multi-step forms
- Upload progress
- Onboarding flows

## Core idea
A progress bar is just a container with a child element whose width
represents progress (usually a percentage).

## Key concepts involved
- Percentage-based widths
- State driving UI
- CSS transitions for smooth updates

## Common mistakes
- Hardcoding widths
- Forgetting smooth transitions
- Tying UI directly to logic without separation
