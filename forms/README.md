## Forms & User Input

This folder covers UI patterns and techniques for building
usable, accessible, and scalable forms.

Forms are one of the most critical UI components and often
involve state management, validation, and user feedback.

### Covers
- Multi-step and wizard forms
- Input styling patterns
- Labels, placeholders, and hints
- Validation feedback and errors
- Progress indicators for form completion

Each subfolder focuses on one form pattern or technique.
